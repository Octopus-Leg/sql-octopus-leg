## 스터디 목표

`SQL 문어발🐙` 스터디는 **정해진 학습 자료(책)를 기반으로 하지만, 이에 멈추지 않고 스터디원 각자 다양한 유형의 문제를 직접 해결하여 공유하는 것**을 목표로 합니다.


## 스터디 진행 방식

- 매주 **정해진 분량의 학습과 유형별 SQL 문제 풀이 및 리뷰**를 중심으로 진행됩니다.

- **주간 진행:**
    - 1️⃣ 매주 정해진 **책의 유형**을 읽고 핵심 개념을 파악합니다.
    - 2️⃣ 해당 챕터의 **유형별 SQL 연습 문제**를 직접 풀이합니다.
    - 3️⃣ 개인의 **문제 풀이 코드를 레포지토리에 커밋**합니다.
    - 4️⃣ 스터디 시간에 각자 풀이한 내용을 발표하고, 이에 대한 질의응답 및 자유로운 토론을 진행합니다.
- **발표:** 각 스터디원은 준비한 내용을 **10~15분** 내외로 발표합니다. 발표 후에는 충분한 질의응답 시간을 가집니다.
